# X² Deploy Station 🚀

一个现代化的机器人部署和管理工具 🤖

## 项目介绍 📖

X² Deploy Station 是一个专门为机器人项目设计的部署管理平台。它提供了直观的图形界面，让用户可以轻松地管理不同版本的机器人，进行部署、更新和运行控制。

### 主要特性 ✨

- 🎯 直观的图形用户界面
- 🔄 自动版本管理和更新
- 🌐 GitHub 集成支持
- 💻 虚拟环境隔离
- 📊 实时日志显示
- ⚙️ 可配置的部署选项
- 🔒 安全的进程管理
- 🌈 自定义主题支持

## 技术栈 🛠️

- Python 3.8+
- CustomTkinter (现代化UI)
- asyncio (异步支持)
- GitHub API (版本控制)
- FastAPI (后端服务)
- SQLite (数据存储)

## 快速开始 🚀

### 环境要求

- Python 3.8 或更高版本
- Git
- Windows/Linux/MacOS 支持
